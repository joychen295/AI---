# 辰极智脑AI中台 · 核心功能汇报

---

## 一、产品定义

**辰极智脑AI中台**是一个面向MBSE行业的企业级AI服务基础设施，支持知识资产管理、智能检索问答、服务治理监控与统一安全认证，为公司全产品线提供标准化AI能力供给。

---

## 二、核心功能

### 功能一：企业知识中枢

| 维度               | 内容                                                                                                                                                                                                                                                                              |
| ------------------ | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **功能名称** | 企业知识中枢                                                                                                                                                                                                                                                                      |
| **功能描述** | 构建覆盖MBSE领域的多层次知识库体系，包括本体论知识（SysML规范、MBSE理论）、方法论知识（内部建模方法论、最佳实践）、案例库（历史项目模型）及外部知识库（合作伙伴行业知识）。支持本地部署与云端托管两种知识引擎模式，具备文档自动解析、向量化索引、知识图谱构建等完整知识生产能力。 |
| **应用场景** | ① 新员工入职培训：快速查阅MBSE规范与建模方法论 `<br>`② 项目复用：检索历史项目模型作为参考模板 `<br>`③ 规范查询：自动化查询行业合规标准                                                                                                                                     |
| **价值量化** | ・知识覆盖率≥80%`<br>`・新员工培养周期缩短50%以上 `<br>`・相似项目建模效率显著提升                                                                                                                                                                                           |

### 功能二：文档资产管理

| 维度               | 内容                                                                                                                                                                                                                                                  |
| ------------------ | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **功能名称** | 文档资产管理                                                                                                                                                                                                                                          |
| **功能描述** | 企业级非结构化文档统一管理平台，支持PDF、Word、Excel、PowerPoint、Markdown、HTML等14种文档格式及8种图片格式的上传存储。文档上传后自动触发解析、分块、向量化、索引构建的完整处理流程，实现非结构化数据的结构化转换。支持文档分类、标签管理、权限控制。 |
| **应用场景** | ① 技术文档归档：军工/航空航天企业海量技术文档统一管理 `<br>`② 项目知识沉淀：项目交付后技术资料系统化存储 `<br>`③ 智能检索：模糊搜索与精准筛选快速定位文档                                                                                      |
| **价值量化** | ・文档查找效率大幅提升 `<br>`・知识资产规范化沉淀，保值增值 `<br>`・支持14种文档格式 + 8种图片格式                                                                                                                                                |

### 功能三：智能检索与问答

| 维度               | 内容                                                                                                                                                                                                                                                                                                           |
| ------------------ | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **功能名称** | 智能检索与问答                                                                                                                                                                                                                                                                                                 |
| **功能描述** | 提供检索型与对话型两种交互模式。检索模式强调单次精准问答与来源可追溯；对话模式支持多轮对话与上下文记忆。支持本地、全局、混合、朴素四种检索策略：本地模式精确查找速度快；全局模式基于知识图谱跨文档关联分析；混合模式兼顾精准与全面；朴素模式适用于简单问答。每次检索结果均提供来源引用、置信度评估与性能指标。 |
| **应用场景** | ① 技术咨询：工程师通过对话即时获取MBSE规范解释与操作指导 `<br>`② 信息检索：大型项目中快速定位关键技术文档内容 `<br>`③ 合规校验：查询行业规范确保设计方案合规                                                                                                                                            |
| **价值量化** | ・知识检索召回率≥85%`<br>`・知识检索精确率≥90%`<br>`・知识库查询延迟<3秒 `<br>`・答案附带来源引用，信息可靠性保障                                                                                                                                                                                      |

### 功能四：服务治理与监控（辰极脉络）

| 维度               | 内容                                                                                                                                                                                                                                                                                          |
| ------------------ | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **功能名称** | 辰极脉络（服务治理与监控）                                                                                                                                                                                                                                                                    |
| **功能描述** | 微服务治理平台，实现服务自动发现与注册（启动自动注册、宕机自动摘除）。以网络拓扑图可视化展示服务间调用关系：节点颜色反映健康度（绿色=健康，红色=故障，黄色=警告），连线粗细反映流量大小，支持点击查看详情或高亮上下游关联。实时监听服务事件，核心服务下线自动触发告警，记录完整操作审计日志。 |
| **应用场景** | ① 日常运维：一站式监控全平台AI子服务运行状态 `<br>`② 故障排查：通过依赖关系可视化快速定位故障源 `<br>`③ 容量规划：基于流量数据进行资源配置优化                                                                                                                                         |
| **价值量化** | ・运维效率显著提升 `<br>`・故障恢复时间大幅缩短 `<br>`・系统可用性≥99%                                                                                                                                                                                                                   |

### 功能五：统一认证与权限

| 维度               | 内容                                                                                                                                                                                                                                                              |
| ------------------ | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **功能名称** | 统一认证与权限                                                                                                                                                                                                                                                    |
| **功能描述** | 与中央管理系统集成，实现单点登录（SSO）与权限统一管理。支持租户级数据隔离，确保不同企业间数据完全独立。提供分级权限控制：平台管理员管理所有租户生命周期与全局系统配置；企业管理员管理本企业业务数据与租户级资源监控；普通用户仅可访问个人业务统计与基础服务状态。 |

| **价值量化** | ・军工级数据安全保障 `<br>`・租户级数据完全隔离 `<br>`・管理效率提升，账号统一管理                                                                                                                                                                            |

### 功能六：系统监控仪表盘

| 维度               | 内容                                                                                                                                                                                                                                                                                                |
| ------------------ | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **功能名称** | 系统监控仪表盘                                                                                                                                                                                                                                                                                      |
| **功能描述** | 实时可视化展示平台关键运行指标。系统健康度采用木桶原则计算，取最低分服务作为整体健康度。监控维度包括：服务存活率（在线服务数/总服务数）、资源使用率（CPU/内存/GPU显存/GPU温度）、业务统计（文档总数、查询总数、存储总量）。支持按角色展示差异化视图，管理员可查看全量详情，普通用户仅见红绿灯状态。 |
| **应用场景** | ① 运维监控：实时掌握系统健康状态，主动发现潜在风险 `<br>`② 资源管理：监控GPU/CPU资源使用，指导资源扩容决策 `<br>`③ 业务分析：统计文档量、查询量等业务指标，支撑运营决策                                                                                                                      |
| **价值量化** | ・API响应时间(P95)<500ms `<br>`・并发用户支持≥100人 `<br>`・运维前置化，问题早发现早处理                                                                                                                                                                                                       |
