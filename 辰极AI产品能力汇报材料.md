## 一、产品体系概述

辰极公司在MBSE（模型驱动系统工程）领域深耕多年，现已形成以**辰极智脑AI中台**为核心，**摇光小智**与**玑衡自绘**为典型应用的AI赋能产品体系。该体系旨在解决传统MBSE软件**学习成本高、建模效率低、知识获取难**三大行业痛点，推动工程软件从"工具使用"向"智能辅助"转型升级。

### 产品定位与关系

| 产品名称               | 定位                 | 目标用户               |
| ---------------------- | -------------------- | ---------------------- |
| **辰极智脑中台** | 公司级AI服务基础设施 | 全产品线技术底座       |
| **摇光小智**     | 需求管理系统AI助手   | 需求分析师、系统工程师 |
| **玑衡自绘**     | 建模工具AI助手       | 建模工程师、系统架构师 |

## 二、辰极智脑AI中台功能概览

辰极智脑AI中台是公司AI战略的**核心基础设施**，承担着统一AI能力供给、知识资产沉淀、智能服务编排三大使命。

### 2.1 企业知识中枢

#### 功能要点

**知识库体系建设**：构建覆盖MBSE领域的多层次知识库体系，包括本体论知识（SysML规范、MBSE理论）、方法论知识（建模最佳实践）、案例库（历史项目模型）三大模块。

**知识检索引擎**：提供检索型与对话型两种交互模式的智能检索服务，支持本地、全局、混合、朴素四种检索策略，满足不同场景下的知识获取需求。

**知识库配置与管理**：支持本地部署与云端托管两种知识引擎模式，具备文档自动解析、向量化索引、知识图谱构建等完整的知识生产能力。

#### 解决的行业问题

| 行业痛点                                                       | 解决方案                                              | 价值体现                  |
| -------------------------------------------------------------- | ----------------------------------------------------- | ------------------------- |
| MBSE领域知识分散，工程师培养周期长（平均6-12个月掌握专业工具） | 构建覆盖率80%以上的行业知识库，提供即时检索与智能问答 | 新员工培养周期缩短50%以上 |
| 历史项目经验难以复用，相似系统重复建模                         | 建立模型资产知识图谱，支持相似模型检索与参考          | 建模效率提升显著          |
| 规范标准分散，合规性校验依赖人工经验                           | 沉淀行业规范至知识库，支持自动化规范查询              | 降低合规风险              |

---

### 2.2 文档资产管理

#### 功能要点

**多格式文档统一管理**：支持PDF、Word、Excel、PowerPoint、Markdown、HTML等14种文档格式及8种图片格式的上传、存储与管理。

**自动化处理流水线**：文档上传后自动触发解析、分块、向量化、索引构建的完整处理流程，实现非结构化数据的结构化转换。

**精细化分类体系**：支持文档分类、标签管理、权限控制，实现企业知识资产的规范化组织与个性化检索。

#### 解决的行业问题

| 行业痛点                                                   | 解决方案                                       | 价值体现         |
| ---------------------------------------------------------- | ---------------------------------------------- | ---------------- |
| 军工、航空航天企业技术文档数量庞大、格式多样，难以有效利用 | 提供统一存储与自动索引，支持模糊搜索与精准筛选 | 文档查找效率提升 |
| 项目交付后的技术沉淀分散，缺乏系统性管理                   | 建立企业级文档中心，实现知识资产规范化沉淀     | 知识资产保值增值 |

---

### 2.3 智能检索与问答

#### 功能要点

**双模式交互**：提供**检索模式**（单次精准问答）与**对话模式**（多轮对话探索）两种交互形态。

**多策略检索**：

- **本地模式**：基于文档块相似度检索，适用于查找特定文档内容
- **全局模式**：基于知识图谱关系推理，适用于跨文档关联分析
- **混合模式**：结合本地与全局策略，兼顾精准与全面
- **朴素模式**：基础相似度检索，适用于简单问答场景

**结果可追溯**：每次检索结果均提供来源引用、置信度评估、性能指标，确保答案的可验证性与透明度。

#### 解决的行业问题

| 行业痛点                                   | 解决方案                                     | 价值体现       |
| ------------------------------------------ | -------------------------------------------- | -------------- |
| 大型系统工程项目文档量大，关键信息查找耗时 | 智能语义检索替代关键词匹配，精准定位所需信息 | 检索效率提升   |
| MBSE规范理解门槛高，工程师需反复查阅手册   | 对话式问答即时获取专业解释与操作指导         | 学习成本降低   |
| 军工行业对信息准确性要求极高               | 答案附带来源引用与置信度，支持人工校验       | 信息可靠性保障 |

---

### 2.4 服务治理与监控

#### 功能要点

**辰极脉络服务治理**：提供微服务注册中心与治理平台，实现服务自动发现、状态监控、依赖分析与故障诊断。

**服务拓扑可视化**：以网络拓扑图展示服务间调用关系，节点颜色反映健康度，连线粗细反映流量大小，辅助快速定位系统瓶颈。

**智能诊断与告警**：实时监听服务事件，当核心服务下线时自动触发告警并记录日志，支持故障根因分析。

#### 解决的行业问题

| 行业痛点                           | 解决方案                           | 价值体现         |
| ---------------------------------- | ---------------------------------- | ---------------- |
| AI中台涉及多个子服务，运维复杂度高 | 统一服务治理平台，一站式监控与管理 | 运维效率提升     |
| 服务故障传递路径难以追溯           | 依赖关系可视化，快速定位故障源     | 故障恢复时间缩短 |
| 系统稳定性要求高（可用性≥99%）    | 自动健康检测与告警机制             | 系统可靠性保障   |

---

### 2.5 系统集成与安全

#### 功能要点

**统一认证体系**：与中央管理系统（ts_manage）集成，实现单点登录（SSO）、权限统一管理、租户隔离。

**分级权限控制**：区分平台管理员、企业管理员、普通用户三级权限体系，确保数据访问的安全边界。

**系统仪表盘**：实时展示系统健康度、服务存活率、资源使用率（CPU/内存/GPU）等关键指标，支持分角色视图。

#### 解决的行业问题

| 行业痛点                             | 解决方案                             | 价值体现     |
| ------------------------------------ | ------------------------------------ | ------------ |
| 军工企业对数据隔离与访问控制要求严格 | 租户级数据隔离，细粒度权限控制       | 合规性保障   |
| 大型企业多系统并行，账号管理复杂     | 对接统一认证中心，一套账号访问多系统 | 管理效率提升 |
| 系统运行状态不透明，问题发现滞后     | 实时仪表盘监控，主动发现潜在风险     | 运维前置化   |

## 三、摇光小智（瑶光+AI对话）功能概览

摇光小智是嵌入瑶光需求管理系统的AI智能助手，专注于需求分析与知识问答，提升需求管理阶段的工作效率。

### 3.1 智能需求问答

#### 功能要点

**项目上下文感知**：通过MCP协议实时获取当前项目的需求数据，提供针对性的分析与建议。

**多源信息融合**：整合本地知识库、外部知识库、联网搜索三大信息来源，确保回答的全面性与准确性。

**多轮对话记忆**：支持连续提问与追问，维护对话上下文，实现深度知识探索。

#### 解决的行业问题

| 行业痛点                                 | 解决方案                               | 价值体现         |
| ---------------------------------------- | -------------------------------------- | ---------------- |
| 需求分析依赖资深工程师经验，知识传承困难 | AI助手沉淀专家经验，随时可查询         | 知识共享效率提升 |
| 需求评审耗时长，问题发现滞后             | 智能问答辅助需求分析，提前发现潜在问题 | 评审效率提升     |
| 项目需求量大，信息检索困难               | 结合项目上下文的智能检索               | 信息获取效率提升 |

---

### 3.2 需求合规性校验

#### 功能要点

**自动化合规检测**：基于行业规范与最佳实践，自动识别需求文档中的合规性问题。

**多维度校验**：覆盖需求完整性、一致性、可追溯性、符合性等多个维度。

**结果可追溯**：校验结果附带具体问题位置与改进建议，便于针对性修正。

#### 解决的行业问题

| 行业痛点                                           | 解决方案                               | 价值体现         |
| -------------------------------------------------- | -------------------------------------- | ---------------- |
| 军工、航空航天行业需求合规要求严格，人工审核效率低 | AI自动化校验，准确率≥85%，误报率<5%   | 合规审核效率提升 |
| 需求问题发现滞后，导致下游设计返工                 | 在需求阶段即实施合规检测，前置问题发现 | 返工成本降低     |
| 不同项目合规标准不统一                             | 可配置的校验规则体系                   | 标准化程度提升   |

## 四、玑衡自绘（玉衡+AI建模）功能概览

玑衡自绘是集成于天玑/玉衡建模工具的AI辅助建模能力，实现自然语言到SysML模型的智能转换。

### 4.1 自然语言建模

#### 功能要点

**文本到模型生成**：用户输入自然语言描述，系统自动生成符合SysML规范的模型结构。

**模型增删改查**：支持通过自然语言指令对模型进行创建、修改、删除、查询操作。

**XMI格式导出**：生成的模型可导出为XMI标准格式，无缝导入天玑/玉衡建模工具。

#### 解决的行业问题

| 行业痛点                                  | 解决方案                 | 价值体现         |
| ----------------------------------------- | ------------------------ | ---------------- |
| SysML建模学习曲线陡峭，专业人员培养周期长 | 自然语言交互降低建模门槛 | 人员培养成本降低 |
| 手工建模效率低，中等复杂度系统建模需2-4周 | AI辅助建模，效率提升10倍 | 项目周期缩短     |
| 建模工具操作繁琐，用户体验差              | 对话式交互，简化操作流程 | 用户满意度提升   |

---

### 4.2 MBSE Copilot（规划中）

#### 功能要点

**知识问答**：基于四层知识库（本体论、方法论、案例库、外部知识）提供MBSE领域专业问答。

**代码/模型补全**：在SysML v2.0文本化建模过程中，实时提供补全建议，提升建模效率。

**智能建议**：分析当前模型结构，提示可能遗漏的元素、不合理的关系、潜在的优化点。

#### 规划价值（26年6月MVP交付）

| 能力维度 | 预期效果                         |
| -------- | -------------------------------- |
| 建模效率 | 从2-4周缩短至2-3天，效率提升10倍 |
| 人力成本 | 单项目减少2-3名建模工程师        |
| 项目周期 | 交付周期从6个月缩短至3-4个月     |

## 七、总结

辰极公司以**辰极智脑AI中台**为核心基础设施，通过**摇光小智**与**玑衡自绘**两大应用产品，构建了完整的MBSE智能化解决方案体系。该体系直击行业痛点，在**人才培养、建模效率、知识管理、协作能力、合规校验**五大维度提供显著价值提升，预计在27年底实现5000万元营收目标。

更重要的是，通过知识资产沉淀、技术能力积累、客户关系建立、生态平台培育四大维度的持续投入，辰极正在构筑难以复制的行业壁垒，抢占AI+MBSE赛道制高点，力争成为该领域的标杆企业与行业标准引领者。
